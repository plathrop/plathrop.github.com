Getting Started

  • Each presentation is a single html file
  • Each slide is defined by a section: 
    
<section>
  <!-- Slide content goes here -->
</section>
  • You can incrementally display content using fragments: 
    
<li class="fragment">
  This list item will display incrementally
</li>
  • Code highlighting is included!

Example Slide


<section>
  <h2>Example Slide</h2>
  <pre class="fragment"><code data-trim>
